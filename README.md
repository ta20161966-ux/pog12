# Weather Diary

**Автор:** Иван Иванов

**Описание:** Простое приложение для ведения дневника погоды. Позволяет добавлять, просматривать и фильтровать записи, сохранять данные в JSON.

**Как запустить:**
1. Установите Python 3.8+.
2. Установите зависимости:
   pip install tkcalendar
3. Запустите:
   python weather_diary.py

**Примеры использования:**
- Добавьте запись с датой, температурой, описанием и отметкой осадков.
- Используйте фильтры по дате и температуре.
- Сохраняйте и загружайте данные через меню.
