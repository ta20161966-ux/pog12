# 📅 Weather Diary

**Автор:** [Ваше Имя Фамилия]

**Описание:**
Простое кроссплатформенное приложение для ведения дневника погоды. Позволяет добавлять, просматривать, фильтровать и сохранять записи о погоде в файл формата JSON. Интерфейс реализован на Tkinter.

---

## ⚙️ Системные требования
- Python 3.8 или новее.
- Библиотека `tkcalendar` для удобного выбора даты.

---

## 🛠️ Установка и запуск

1. **Установите Python**
   Скачайте с официального сайта: https://www.python.org/downloads/

2. **Установите зависимости**
   Откройте терминал (командную строку) в папке проекта и выполните:
   ```bash
   pip install tkcalendar
   ```

3. **Запустите приложение**
   ```bash
   python main.py
   ```

---

## 📝 Как пользоваться

1. **Добавление записи**
   - Введите дату (можно выбрать в календаре).
   - Укажите температуру (число).
   - Добавьте описание (например, *«Солнечно»*).
   - Поставьте галочку, если были осадки.
   - Нажмите **«Добавить запись»**.

2. **Фильтрация**
   - Введите дату в поле «Фильтр по дате», чтобы увидеть записи только за этот день.
   - Введите температуру в поле «Фильтр по температуре (выше)», чтобы увидеть записи, где температура выше указанного значения.

3. **Сохранение и загрузка**
   - Меню **«Файл»** → **«Сохранить»** или **«Загрузить»** для работы с файлом `data.json`.
